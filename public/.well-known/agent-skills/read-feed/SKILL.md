# Read Feed Skill

This skill allows agents to fetch, digest, and search posts published on alexgetman.com by Alex Getman.

## How to use
1. Fetch the JSON feed array from `https://alexgetman.com/feed.json`.
2. To read posts in a structured format, iterate through the `items` array.
3. Present the relevant posts to the user.
